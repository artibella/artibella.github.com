Hi there!

My name is Richard Bausek and I'm an interactive developer passionate about user experience, interface and interaction design.

About me

Crafting code for the web since 2000 I have gathered experience working with a wide range of web technologies, specializing in frontend development with web standards (HTML5, CSS3, JavaScript) and the Flash platform (Flash, Flex, AIR) for desktop and mobile environments. The last four and a half years I have been working as a senior developer for Moccu in Berlin.

Seeking new challenges outside the world of web agencies, I’ll be available for full time work again starting 2012. If you happen to be building cutting edge products, services or applications and in need of an experienced frontend developer then please don’t hesitate to contact me.

When not working on projects, I enjoy experimenting with new tools and technologies, help run the Rotaug artist collective and suck at playing the bass guitar.
